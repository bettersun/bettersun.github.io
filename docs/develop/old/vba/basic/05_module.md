# 5. 模块

## 5.1 模块种类

> VBA模块有四类：
>   
> * Microsoft Excel对象
> * 窗体
> * 模块
> * 类模块

> 如下图：
 
![](pic/05_01_module.png)

## 5.2 操作

> 可在工程窗口点击右键在菜单中插入、删除、导入、导出各类模块  
> 选择模块后(窗体模块需要双击)，点击属性，可以修改对应的名称

> 模块导出文件类型： 
>  
> * 模块 : *.bas
> * 窗体 : *.frm
> * 类模块 : *.cls
